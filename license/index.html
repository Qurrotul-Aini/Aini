    METODE NEWTON RAPSHON¶

    Pengertian Metode Newton Rapshon¶

    ​ Metode newton rapshon adalah suatu metode untuk mencari akar dari suatu fungsi f(x) dengan menggunakan pendekatan satu titik, dimana fungsi f(x) mempunyai turunan f'(x) dengna formula,

    f^{n} = \frac{df(x)}{dx}

    Catatan : Semakin dekat titik awal (x0) dengan akar sebenarnya, maka semakin cepat konvergen ke akarnya.

    Penurunan Rumus Iterasi Newton-Raphson¶

    ​ Iterasi Newton-Rapshon berawal dari sebuah hampiran awal untuk akar r, kemudian menghitung hampuran selanjutnya dengan cara sebagai berikut.

    • Misalkan Xn adalah hampiran awal pada langkah ke-n, dimana n=0,1,2,.....

    • Hitunglah gradien garis singgung terhadap kurva y = f (x) di titik (Xn,f(Xn)) yakni f'(Xn) dan tentukan persamaan garis singgungnya , yakni

    y=f^{1}(x_{n})(x-x_{n})+f(x_{n})
    • Hampiran berikutnya adalah absis titik potong garis singgung tersebut dengan sumbu-x, yakni
    x_{n+1} = x_{n} - \frac {f(x_{n})}{f^{1}(x_{n})}

    Formula iterasi juga dapat diturunkan dari deret Taylor f(x) di sekitar Xn, yakni:

    f(n) = f(x_{n}) + (x-x_{n})f^{1}(x_{n})+\frac{1}{2}(x-x_{n})^{2}f^{n}(x_{n})+...+\frac{1}{n}(x-x_{n})^{n}f^{n}(x_{n})
    Algoritma Metode Newton-Rapshon¶

    ​ Berikut algoritma dari metode Newton-Rapshon:

    1. Definisikan fungsi f(x) dan f'(x)
    2. Tentukan toleransi error (e) dan iterasi maksimum (n)
    3. Tentukan nilai pendekatan awal (x0)
    4. Hitung f(x0) dan f'(x0)
    5. Untuk iterasi i=1 sampai dengan |f(xi)|>=e, dimana akar persamaanya diperoleh dari nilai xi terakhir, berikut formulanya
    x_{n+1}=x_{n} - \frac {f(x_{n})}{f^{1}(x_{n})}
    Program Newton-Rapshon pada Fungsi Persamaan Non-Linear¶
    import math
e = 2.71828

def fungsi(x):
    x = float((e**x) - (4*x))
    return x

def fungsiturunan(x):
    x = float((e**x) - (4))
    return x

x = float(input('Masukkan nilai awal = '))
error = float(input('Masukkan nilai error = '))
perulangan = int(input('Masukkan maksimal pengulangan = '))

iterasi = 0
selisih = error+1
while iterasi <= perulangan  and selisih>error :
    iterasi += 1
    f_2 = x - (fungsi(x)/fungsiturunan(x))
    selisih = math.fabs(f_2 - x)
    x = f_2
    print("Iterasi ke = ",iterasi,", x = ",f_2, ", f(",f_2,") = ",fungsi(f_2),", selisih = ",error)
    if iterasi <= perulangan:
        print("Perulangan Mencapai Batas Maksimal dengan hasil = ", f_2)
    else :
        print("Toleransi tidak terpenuhi")
